I am a proponent of resource sharing, open source, and data autonomy.

Creating your personal open-source website is not only a form of artistic expression; it is also a form of resistance. You own your domain, your website, your words, your data. You are using your technology as a tool; no one behind the screen is using you.

This is an abridged version of a website-making workshop that I hosted for my buddies to create their own free, open-source websites to share their work with the world. You can read more in my original LinkedIn post.

Here’s a tutorial to learn about GitHub and started building your site:

Website Creation Workshop

Step 1: Create a GitHub Account

  • - Supplemental Slides: View PDF  |  Download PDF
  • - Go to GitHub and sign up for an account.
  • - Verify your email address.

Step 2: Download an HTML and CSS template or use Jekyll

  • - Visit a site such as HTML5 UP or Jekyll.
  • - Choose a template you like and click "Download".
  • - Extract the downloaded .zip file on your computer.

Step 3: Create a New GitHub Repository

  • - Log in to GitHub and click the + icon (top right), then select New repository.
  • - Name it yourname.github.io (replace yourname with your GitHub username).
  • - Set it to Public and click Create repository.

Step 4: Upload the Website Files to GitHub

  • - In your new repository, click Upload files.
  • - Drag and drop all extracted files from your HTML5 UP template.
  • - Click Commit changes to save.

Step 5: Enable GitHub Pages

  • - Go to Settings > Pages.
  • - Under Branch, select main and click Save.
  • - Your site will be available at https://yourname.github.io/ in a few minutes!

Step 6: Customize Your Website

  • - Edit the index.html file directly in GitHub or using a code editor like VS Code.
  • - Modify the text, links, and images to personalize your site.
  • - Commit changes to update your website live.

Tips

  • - AI tools such as ChatGPT are a great resource to generate code for your site. For this website for example, I took a template and then edited it by producing ChatGPT code for elements such as formatting, resume download, and identifying bugs.
  • - Listen to some of your favorite tunes as you work on creating your dream website! I'm a huge NPR Tiny Desk fan myself.
  • - Look at some other websites for inspiration.
